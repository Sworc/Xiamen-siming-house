# - 厦门思明区二手房价分析和预测
基于Python：网络爬虫获取房价信息，数据的预处理和可视化，搭建机器学习模型，预测房价。

### -数据来源：房天下
- 项目背景：开始之前，先讲个网上看到的一条评论，某地房价5w每平，而我仅仅月入过万，掐指一算，命中注定买房是不可能的，这辈子都不可能买房，感叹：“我真的还想再活500年······”。 房子虽然很贵买不起，但是可以应用科学的方法了解房价趋势，做到心中有数，万一买的起呢？
- 项目内容：
    - 1、利用网络爬虫获取厦门市思明区二手房信息
    - 2、进行数据预处理
    - 3、进行数据可视化
    - 4、构建机器学习房价预测模型
    - 5、预测房价
